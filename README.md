## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Configuration file
**.env** file as follows:
<ul>
<li>DB_TYPE=mysql</li>
<li>DB_HOST=localhost</li>
<li>DB_PORT=3306</li>
<li>DB_USERNAME=</li>
<li>DB_PASSWORD=</li>
<li>DB_DATABASE=</li>
</ul>
